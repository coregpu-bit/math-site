피타고라스의 정리

직각삼각형에서 밑변 a, 높이 b, 빗변 c 사이에는 항상 a² + b² = c² 가 성립합니다.

1. 변을 직접 움직여 보기

삼각형의 빨간 점을 끌거나 슬라이더를 움직여 보세요. 세 변 위에 그려진 정사각형의 넓이가 실시간으로 바뀝니다. a² + b² 의 합이 언제나 c² 와 같다는 것을 확인하세요.

a² = 0  +  b² = 0  =  c² = 0   (c = 0)

2. 넓이를 옮겨 담는 증명

두 작은 정사각형(a² + b²)의 조각들을 큰 정사각형(c²) 안으로 옮겨 보세요. 빈틈없이 꼭 맞게 채워진다면, 두 넓이의 합이 정확히 같다는 뜻입니다.

초록·주황 정사각형의 네 조각이 파란 정사각형으로 이동하며 빈틈없이 채워집니다. → a² + b² = c²
