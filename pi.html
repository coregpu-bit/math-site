원주율 π

원주율은 원의 둘레 ÷ 지름입니다. 원이 크든 작든 이 비율은 언제나 같은 값, π ≈ 3.14159… 가 됩니다. 세 가지 방법으로 직접 확인해 보세요.

1. 원을 굴려서 둘레를 펴 보기

지름이 1인 원을 한 바퀴 굴리면, 굴러간 거리가 곧 둘레입니다. 그 거리가 지름의 몇 배인지 보세요.

굴러간 거리 ÷ 지름 = 0.000 (한 바퀴 다 굴리면 π에 도달)

2. 다각형으로 원에 다가가기

원 안과 밖에 정다각형을 그립니다. 변의 개수 n을 늘릴수록 다각형은 원에 가까워지고, 둘레 ÷ 지름 값이 위·아래에서 π를 좁혀 옵니다. (고대 아르키메데스의 방법!)

안쪽 다각형: 0 <  π ≈ 3.14159…  < 0 :바깥 다각형

3. 다트를 던져 π 구하기 (몬테카를로)

정사각형 안에 무작위로 점을 뿌립니다. 원 안에 떨어진 점의 비율은 원 넓이 ÷ 정사각형 넓이 = π/4 에 가까워집니다. 점을 많이 던질수록 추정값이 π에 수렴합니다.

던진 점 0 · 원 안 0 → π 추정값 = 0.000
