인수분해 — 블록으로 보기

인수분해는 x² + bx + c 같은 식을 (x + p)(x + q) 처럼 곱셈의 형태로 바꾸는 것입니다. 블록(넓이 모형)으로 보면, 흩어진 블록들을 빈틈없는 직사각형으로 맞추는 일과 같습니다. 그 직사각형의 가로·세로 길이가 바로 인수예요.

x²-블록 (한 변이 x인 정사각형) x-블록 (가로 x, 세로 1) 1-블록 (1×1 정사각형)

1. 두 수 p, q 를 정해 직사각형 만들기

가로는 x + p, 세로는 x + q. 슬라이더로 p, q 를 바꾸면 블록들이 직사각형을 채웁니다. 직사각형 전체 넓이가 곧 전개식이고, 가로·세로가 곧 인수입니다.

(x + 2)(x + 3)  =  x² + 5x + 6

2. 직접 인수분해 도전

아래 식 x² + bx + c 를 (x + ?)(x + ?) 로 만들어 보세요. 두 수를 더하면 b, 곱하면 c 가 되는 짝을 찾으면 됩니다. 맞히면 블록이 직사각형으로 완성돼요!
